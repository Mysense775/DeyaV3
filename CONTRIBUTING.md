# Contributing to Deya v3.0 🌺

Спасибо за интерес к проекту! Вот как можно помочь:

## Как Внести Вклад

### 1. Сообщить о Баге
- Используйте [Issues](https://github.com/Mysense775/DeyaV3/issues)
- Опишите шаги воспроизведения
- Укажите версию и окружение

### 2. Предложить Фичу
- Создайте Issue с тегом `enhancement`
- Опишите use case и ожидаемое поведение

### 3. Отправить PR
1. Форкните репозиторий
2. Создайте ветку: `git checkout -b feature/my-feature`
3. Сделайте коммиты: `git commit -m "feat: описание"`
4. Отправьте PR

## Структура Коммитов

```
feat: новая фича
fix: исправление бага
docs: документация
style: форматирование
refactor: рефакторинг
test: тесты
chore: обслуживание
```

## Разработка

### Локальная Настройка

```bash
git clone https://github.com/Mysense775/DeyaV3.git
cd DeyaV3
npm install
pip3 install -r requirements.txt
cp .env.example .env
npm run dev
```

### Тестирование

```bash
npm test
```

### Линтинг

```bash
npm run lint
```

## Code Style

- JavaScript: ESLint + Prettier
- Python: PEP 8
- Коммиты на английском
- Документация на русском и английском
