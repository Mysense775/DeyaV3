# E-commerce Manager Skill

## Назначение
Управление интернет-магазином: товары, цены, заказы, интеграция с маркетплейсами и аналитика продаж.

## Поддерживаемые платформы
### CMS для магазинов
- WooCommerce (WordPress)
- Shopify
- Magento
- OpenCart
- 1С-Битрикс
- PrestaShop

### Маркетплейсы
- Ozon
- Wildberries
- Яндекс.Маркет
- AliExpress
- Amazon
- eBay
- Avito

### Платёжные системы
- ЮKassa
- Robokassa
- Stripe
- PayPal
- CloudPayments

## Возможности
### Управление товарами
- Массовое добавление/обновление
- Категории и атрибуты
- Вариации (размер, цвет)
- Остатки на складах
- Импорт/экспорт (CSV, XML, YML)

### Ценообразование
- Динамическое ценообразование
- Акции и скидки
- Промокоды
- Цены конкурентов (мониторинг)
- Минимальная/рекомендуемая цена

### Заказы
- Обработка заказов
- Статусы и уведомления
- Интеграция с доставкой
- Возвраты
- Отмены

### Аналитика
- Выручка и прибыль
- Конверсия
- Средний чек
- Популярные товары
- ABC/XYZ анализ

## Команды
- `product add <data>` — добавить товар
- `product update <id> <data>` — обновить товар
- `product sync <marketplace>` — синхронизировать с маркетплейсом
- `price set <id> <price>` — установить цену
- `price monitor <competitor>` — мониторить цены конкурентов
- `order list [--status=<status>]` — список заказов
- `order process <id>` — обработать заказ
- `report sales [--period=<period>]` — отчёт по продажам

## Интеграции
- API маркетплейсов
- Службы доставки (СДЭК, Почта России)
- CRM (Битрикс24, amoCRM)
- Складские системы (МойСклад)
- 1С

## Автоматизация
- Обновление остатков
- Переоценка
- Уведомления о низких остатках
- Автоответы на заказы
- Рассылка о брошенных корзинах

## Пример использования
```
product add "{"name":"iPhone 15", "price":99999, "category":"phones"}"
product sync ozon
price set 123 89999
price monitor competitor.com
order list --status=new
order process 456
report sales --period=last_month
```

## Отчёты
- Продажи по дням/неделям/месяцам
- Топ товаров
- Конверсия воронки
- ROI по каналам
