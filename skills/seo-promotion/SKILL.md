# SEO Promotion Skill

## Назначение
Комплексная SEO-оптимизация: аудит сайтов, анализ конкурентов, подбор ключевых слов, построение ссылок и отслеживание позиций.

## Возможности
### Технический аудит
- Проверка скорости загрузки
- Мобильная адаптивность
- Индексация страниц
- Дубли контента
- Битые ссылки
- XML Sitemap
- Robots.txt
- Schema.org разметка

### Контент-анализ
- Плотность ключевых слов
- Уникальность текста
- Читаемость
- Структура заголовков (H1-H6)
- Мета-теги
- Alt-теги изображений
- Внутренние ссылки

### Анализ конкурентов
- Топ-10 по ключевым словам
- Обратные ссылки конкурентов
- Контент-стратегия
- Слабые места

### Подбор ключевых слов
- Частотность (высокая, средняя, низкая)
- Конкурентность
- Seasonality
- Long-tail
- Semantic core

### Построение ссылок
- Гостевые посты
- Каталоги сайтов
- Пресс-релизы
- Social signals
- PBN (осторожно)

### Отслеживание позиций
- Ежедневный мониторинг
- Уведомления об изменениях
- История позиций
- SERP features

## Команды
- `seo audit <url>` — технический аудит
- `seo keywords <topic> [--lang=<lang>]` — подбор ключевых слов
- `seo competitors <keyword>` — анализ конкурентов
- `seo content <url>` — анализ контента
- `seo backlinks <url>` — анализ ссылок
- `seo track <url> <keywords>` — отслеживание позиций
- `seo optimize <url>` — рекомендации по оптимизации

## Инструменты
- Google PageSpeed Insights
- Google Search Console
- Яндекс.Вебмастер
- Ahrefs API
- SEMrush API
- Serpstat API
- Screaming Frog (через API)

## Отчёты
- PDF-аудит
- Excel с ключевыми словами
- Графики позиций
- Рекомендации по действиям

## Пример использования
```
seo audit example.com
seo keywords "искусственный интеллект" --lang=ru
seo competitors "нейросети для текста"
seo content blog.example.com/post-1
seo backlinks example.com
seo track example.com "нейросеть,AI,chatbot"
```
