# Article Writer Skill

## Назначение
Написание качественных статей для блогов, новостных сайтов, Medium, Хабр и других платформ.

## Возможности
- Генерация идей для статей на основе трендов
- SEO-оптимизация: ключевые слова, мета-теги, структура
- Исследование темы через веб-поиск
- Создание структуры статьи (заголовки, подзаголовки)
- Написание текста разных стилей: информационный, развлекательный, экспертный
- Проверка уникальности и читаемости
- Генерация изображений/иллюстраций
- Автопубликация на платформы

## Команды
- `article idea <topic>` — сгенерировать идеи
- `article outline <topic>` — создать структуру
- `article write <topic> [--style=<style>] [--length=<words>]` — написать статью
- `article seo <article>` — оптимизировать для SEO
- `article publish <platform> <article>` — опубликовать

## Стили написания
- `informational` — информационный (новости, гайды)
- `entertaining` — развлекательный (истории, юмор)
- `expert` — экспертный (аналитика, исследования)
- `persuasive` — убедительный (продажи, маркетинг)

## SEO-оптимизация
- Подбор ключевых слов
- Оптимизация заголовков H1-H3
- Мета-описания
- Внутренние/внешние ссылки
- Читаемость (Flesch Reading Ease)

## Интеграции
- WordPress REST API
- Medium API
- Хабр API
- Ghost API
- Dev.to API

## Пример использования
```
article idea "искусственный интеллект"
article outline "Как ИИ меняет маркетинг"
article write "Как ИИ меняет маркетинг" --style=expert --length=1500
article seo "<article_id>"
article publish medium "<article_id>"
```
