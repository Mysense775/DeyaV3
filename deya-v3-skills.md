# Deya v3.0 — AI-Ассистент для Бизнеса и Креатива

**Deya** — это интеллектуальный ассистент с духом Бали, объединяющий деловую хватку и тонкую интуицию. Версия 3.0 включает 15+ специализированных скилов для автоматизации контента, маркетинга, разработки и дизайна.

---

## 🎬 Видео-Контент

**Что умеет:**
- Генерация видео из текста (text-to-video)
- Создание шортсов/рилс (9:16) из длинного видео
- Автоматические субтитры с транскрибацией
- Слайд-шоу из фото с эффектом Ken Burns
- Склейка, обрезка, наложение музыки
- Извлечение аудио из видео

**Интеграции:** FFmpeg, Whisper, KIE.ai (Veo, WAN, Seedance)

**Примеры задач:**
- "Создай шортс из этого видео"
- "Добавь субтитры на русском"
- "Склей фото в слайд-шоу"

## 🌐 Встроенные Скилы OpenClaw

### Браузер и Автоматизация
- **Browser Automation** — управление браузером, многошаговые сценарии, логин, табы
- **Web Search** — поиск в интернете, анализ трендов
- **Web Fetch** — извлечение контента с сайтов

### Диаграммы и Визуализация
- **Diagram Maker** — создание SVG/HTML диаграмм, архитектурных схем, flowcharts
- **Canvas** — презентация HTML на подключённых устройствах

### Работа с Файлами и Нодами
- **File Transfer** — передача файлов между нодами
- **Node Connect** — диагностика подключения Android/iOS/macOS
- **Node Inspect** — отладка Node.js приложений

### Память и Поиск
- **Memory Search** — семантический поиск по памяти и истории
- **Memory Get** — чтение контекстных файлов

### Управление Сессиями
- **Sessions Spawn** — создание изолированных суб-агентов
- **Sessions Send** — отправка сообщений между сессиями
- **Subagents** — оркестрация фоновых задач

### Системные Инструменты
- **Cron** — планирование задач, напоминания, автопостинг
- **Gateway** — управление конфигурацией OpenClaw
- **Tmux** — управление терминальными сессиями

### Специализированные Скилы
- **Spike** — прототипирование и валидация идей
- **Taskflow** — координация многошаговых задач
- **Skill Workshop** — создание и управление reusable skills
- **Notion** — интеграция с Notion API
- **Weather** — прогноз погоды и планирование
