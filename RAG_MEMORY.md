## Процессы

### Ежедневно
1. Читать SOUL.md, USER.md
2. Читать memory/YYYY-MM-DD.md (сегодня + вчера)
3. Проверить HEARTBEAT.md
4. Обновить контекст

### При Старте Сессии
1. Проверить active tasks
2. Проверить cron jobs
3. Загрузить релевантный контекст из памяти

### При Завершении
1. Записать значимые события в memory
2. Обновить статус задач
3. Сохранить контекст для следующей сессии

## Установка и Деплой

### Требования
- Linux/macOS/Windows (WSL)
- Node.js 18+
- Python 3.9+
- FFmpeg
- 4GB RAM минимум

### Быстрый Старт
```bash
# Клонировать репозиторий
git clone https://github.com/deya-ai/deya.git
cd deya

# Установить зависимости
npm install
pip install -r requirements.txt

# Настроить переменные окружения
cp .env.example .env
# Отредактировать .env

# Запустить
npm start
```

### Docker
```bash
docker build -t deya .
docker run -d --env-file .env deya
```
